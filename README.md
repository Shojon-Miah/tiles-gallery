# Tiles Gallery — Discover Your Perfect Aesthetic

A premium web application designed to showcase a curated collection of high-quality porcelain and ceramic tiles. Users can browse the gallery, search for specific styles, view detailed information about each tile, and manage their personal profiles.

## 🚀 Live Demo
[Tiles Gallery Live](https://tiles-gallery-demo.vercel.app) *(Replace with your actual live URL)*

## ✨ Key Features
- **Curated Gallery**: Browse an extensive collection of premium tiles with a high-end design aesthetic.
- **Advanced Search**: Easily find tiles by title using the integrated search functionality.
- **Dynamic Marquee**: Real-time updates and featured arrivals displayed via a scrolling marquee.
- **User Authentication**: Secure login and registration powered by **BetterAuth** with MongoDB integration.
- **Social Login**: Quick access via Google Social Authentication.
- **User Profile Management**: View and update your profile information, including name and photo.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing experiences.
- **Interactive UI**: Smooth transitions and mobile-optimized sliders using **SwiperJS**.

## ⚙️ Environment Variables
To run this project locally, you will need to add the following variables to your `.env.local` file:

```env
MONGODB_URI=your_mongodb_connection_string
BETTER_AUTH_SECRET=your_auth_secret
BETTER_AUTH_URL=http://localhost:3000
NEXT_PUBLIC_BETTER_AUTH_URL=http://localhost:3000
GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret
```

## 🛠️ Local Development

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Shojon-Miah/tiles-gallery.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```
   This will start both the Next.js app and the JSON server concurrently.

4. **Open the app**:
   Navigate to [http://localhost:3000](http://localhost:3000).
